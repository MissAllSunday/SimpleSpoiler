[center][color=purple][size=12pt][font=verdana]SimpleSpoiler[/font][/size][/color][/center]

[color=green][size=12pt]for SMF 2.0.x only[/size][/color]

SimpleSpoiler is a simple modification that allow you to use the [nobbc] [spoiler] [/spoiler] [/nobbc]  tags.

-No settings, just install and use.
-Works with every theme, [color=red]make sure you upload the spoiler.gif file to your theme's image/bbc  folder[/color]
